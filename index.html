Tutorial membuat PCB menggunakan aplikasi Altium
      

      

      Buka aplikasi Altium
      Buat project baru
      Buat dokumen schematic (add new to project) 
      Simpan sesuai nama dokumen dan project 
      Buat skema (simpan)
      Tools - Annotate Schematics - Update Changes List - OK
      Accept Changes (Create ECO) - Validate - Execute - Close
      Buat dokumen PBC (klik home - PCB Design - PCB Documents Wizard - Save)
      Untuk membuat PCB hanya single layer gunakan Auto Route - Edit Layer Directions - Component Side diset Not Used
      Kembali ke schematics (Design - Update PCB Documents - Validate Changes - Execute Changes - Close)
      Buka dokumen PCB ( Komponen sudah diupdate dari schematics)
      Atur tata letak komponen dan Tools - Auto Route (
      Tekan angka 3 untuk tampilan PCB 3D dan angka 2 untuk tampilan PCB 2D
      Klik View - Legacy 3D untuk melihat tampilan 3D
